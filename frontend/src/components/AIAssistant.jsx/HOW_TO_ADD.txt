# How to add AIAssistant to your FlightSight project

## Step 1 - Copy the file
Copy AIAssistant.jsx into:
C:\Users\surya\Downloads\flightsight_deploy\flightsight_v2\frontend\src\components\AIAssistant.jsx

## Step 2 - Update App.jsx
Open frontend/src/App.jsx and add these two lines:

At the top with other imports, add:
  import AIAssistant from './components/AIAssistant'

Inside the Dashboard function, just before the closing </div> of the main layout, add:
  <AIAssistant
    currentFlight={fd?.flight}
    origin={fd?.origin}
    destination={fd?.destination}
    eta={fd?.eta}
  />

## Step 3 - Push to GitHub
cd C:\Users\surya\Downloads\flightsight_deploy\flightsight_v2
git add .
git commit -m "Add AI Assistant"
git push

## Step 4 - Done!
The AI button appears as a floating purple robot button in the bottom right corner of your app.
